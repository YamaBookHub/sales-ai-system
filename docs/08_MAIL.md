# 08_MAIL

## メール設計

営業メールテンプレート、禁止表現、固定自己紹介、固定実績、文体ルールを管理します。

## CAMPFIRE営業メール定型文

件名:

```text
CAMPFIREでのプロジェクトを拝見しご連絡いたしました
```

本文:

```text
【企業名＋ご担当者】 様

お世話になっております。
株式会社第弐ヴォヌールの山本と申します。

CAMPFIREにて、貴社の「【商品名】」を拝見しました。

【商品の魅力・特徴・強み】という点がとても印象的で、
【使う人】にとって、実際の使用シーンをイメージしやすい商品だと感じました。

弊社では、クラウドファンディング支援およびSNSマーケティング支援を行っております。

実績としては、SNS運用で1か月総再生400万回超、
クラウドファンディング領域では、担当商品で3,500万円規模の売上実績がございます。

商品の魅力を伝える見せ方や、売上につながる導線づくりの面でもお手伝いしております。

もし何かお力になれそうな機会がございましたら、
お気軽にご連絡いただけますと幸いです。
```

AIが埋める項目:

- 企業名＋ご担当者
- 商品名
- 商品の魅力・特徴・強み
- 使う人

担当者名が不明な場合は `企業名 ご担当者様` とする。本文では `様` を二重に付けない。

改行ルール:

- 1段落ごとに空行を1つだけ入れる。
- 1文の途中に空行を入れない。
- 読点 `、` の直後で段落を分けない。
- 長文でも意味のまとまりを優先し、テンプレート都合の細切れ改行は避ける。
- 指定された定型文にない過去商品分析、動画アイデア、強い提案、情報交換依頼を追加しない。
